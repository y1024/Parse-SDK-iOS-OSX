PFLogLevel

enum PFLogLevel {}

PFLogLevel enum specifies different levels of logging that could be used to limit or display more messages in logs.

See

Parse.+setLogLevel:

See

Parse.+logLevel
  • PFLogLevelNone

    Log level that disables all logging.

    Declaration

    Objective-C

    PFLogLevelNone = 0

    Swift

    case none = 0
  • PFLogLevelError

    Log level that if set is going to output error messages to the log.

    Declaration

    Objective-C

    PFLogLevelError = 1

    Swift

    case error = 1
  • PFLogLevelWarning

    Log level that if set is going to output the following messages to log:

    • Errors
    • Warnings

    Declaration

    Objective-C

    PFLogLevelWarning = 2

    Swift

    Log level that if set is going to output the following messages to log:

    • Warnings
    • Informational messages

    Objective-C
